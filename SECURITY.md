# Security Policy

## 🔒 Supported Versions

We actively maintain and patch security vulnerabilities in the following plugin versions:

| Plugin         | Supported Versions     |
|----------------|------------------------|
| xLogin         | Latest release only    |
| xAntiCheat     | Latest release only    |
| All others     | Latest release only    |

> We strongly recommend always running the **latest version** of every xPlugins plugin.

---

## 🚨 Reporting a Vulnerability

**Please do NOT report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability in any xPlugins plugin, please report it responsibly through one of the following channels:

### Option 1 — GitHub Private Security Advisory (Preferred)
Use GitHub's built-in security reporting:
1. Go to the affected plugin's repository
2. Click **Security** → **Advisories** → **Report a vulnerability**

### Option 2 — Discord (Direct Message)
Contact the founder directly on Discord:
- **Server:** [xPlugins Discord](https://discord.com/invite/mNdxNVx6cU)
- **Contact:** `jn_willfrydev`

---

## 📋 What to Include in Your Report

To help us address the issue quickly, please provide:

- **Plugin name and version** affected
- **Minecraft / server version** where the vulnerability was found
- **Description** of the vulnerability and its potential impact
- **Steps to reproduce** the issue
- **Proof of concept** (if available) — screenshots, logs, or code snippets
- Your **suggested fix** (optional but appreciated)

---

## ⏱️ Response Timeline

| Stage                        | Target Time     |
|------------------------------|-----------------|
| Acknowledgement of report    | Within 48 hours |
| Initial assessment           | Within 5 days   |
| Fix released (if confirmed)  | Within 14 days  |

---

## 🙏 Responsible Disclosure

We kindly ask that you:

- Give us reasonable time to investigate and patch before any public disclosure
- Avoid exploiting the vulnerability beyond what is necessary to demonstrate it
- Do not disclose details publicly until a fix has been released

We appreciate responsible security researchers and will credit you in the release notes if you wish.

---

> 🔐 Security is a top priority for xPlugins, especially for plugins like **xLogin** and **xAntiCheat**. Thank you for helping us keep our community safe.
